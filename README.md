# Azure Infra Lab 🌐

## 🔹 Descrição do Projeto

Este é um laboratório de **infraestrutura na Azure** usando **Terraform**, criado para aprendizado e prática de provisionamento automatizado.  
O projeto organiza scripts, diagramas e documentação para facilitar o entendimento da infraestrutura.

**Objetivos:**
- Provisionar máquinas virtuais no Azure
- Configurar redes, sub-redes e recursos básicos
- Automatizar scripts de inicialização (PowerShell / Bash)
- Documentar a arquitetura e outputs

> Nota: Nenhuma credencial ou arquivo sensível está incluído no repositório.
